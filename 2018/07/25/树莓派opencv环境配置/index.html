树莓派opencv环境配置

发表于 2018-07-25 | 更新于: 2018-07-25

主要思路:

在网上下载opencv的源码包进行本地的编译

一、安装opencv编译器

安装opencv所要用到的软件主要有几个：cmake、git、pkg-config、gcc。

1、Cmake

CMake是一个跨平台的安装(编译)工具,可以用简单的语句来描述所有平台的安装(编译过程)。他能够输出各种各样的makefile或者project文件。

总的来说，cmake可以生成Makefile文件，单系统执行编译命令的时候可以按照Makefile文件进行编译（可以认为是编译的说明书(#^.^#)）

2、Git

Git是一个开源的分布式版本控制系统。分布式相比于集中式的最大区别在于开发者可以提交到本地，每个开发者通过克隆（git clone），在本地机器上拷贝一个完整的Git仓库。

可以利用这个命令方便的从GitHub上面clone一些优质的开源项目的源代码。比如说可以从GitHub上Clone opencv的源代码。

3、Pkg-config

pkg-config 是一个在源代码编译时查询已安装的库的使用接口的计算机工具软件.

4、Build-essential

Build-essential是一个编译软件。在进行软件的编译，需要gcc/g++等，都需要用到这个。当然也可以分别单独安装gcc/g++的软件包，用build-essential 只不过方便管理。

5、GCC

gcc是原名为GNU C语言编译器（GNU C Compiler），因为它原本只能处理C语言。GCC很快地扩展，变得可处理C++。之后也变得可处理Fortran、Pascal、Objective-C、Java、Ada，以及Go与其他语言。

简而言之，GCC是一个把c语言变成二进制机器语言的一个“翻译工具”。

综上

输入命令sudo apt install gcc cmake git pkg-config

二、安装相应的OpenCV图像工具包

1、安装JPEG图像工具包sudo apt install libjpeg8-dev

2、安装TIFF图像工具包sudo apt-get install libtiff5-dev

3、安装Jasper图像工具包sudo apt-get install libjasper-dev

4、安装PNG图像包sudo apt-get install libpng12-dev

三、安装视频IO包

1、安装FFmpeg视频图像处理
sudo apt-get install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev

四、安装GTK2.0及其优化包

GIMP（名称由 GNU Image Manipulation Program（GNU图像处理程序）的首字母组成）是一个自由及开放原始码的点阵图图像编辑器，用于图像照片润饰及编辑、自由绘图、调整大小、裁剪、相片蒙太奇、装换图像格式以及其他专业任务。

而GTK+最初是GIMP的专用开发库（GIMP Toolkit），后来发展为Unix-like系統下开发图形界面的应用程序的主流开发工具之一。GTK+是自由软件，并且是GNU计划的一部分。

1、安装GTK2.0sudo apt install libgtk2.0-dev

2、安装优化包sudo apt-get install libatlas-base-dev gfortran

五、下载OpenCV的源码包

，

方法一：官网下载

点击进入OpenCV官网，选择sources下载。注意下载的文件一定是源码包，而不是.EXE文件。版本可以自己选择

方法二：从GitHub上下载

在Linux下找一个文件夹下载git clone https://github.com/opencv/opencv.git

方法三：从国内找百度云资源

复制链接下载`https://pan.baidu.com/s/17STKxlNZDevFafSh0hwuFA

六、编译OpenCV

1、进入相应的opencv下载目录，ps：记得先解压

2、创建release文件夹 mkdir release

3、进入release文件夹 cd ./release

4、cmake读取所有源码，生成Makefile
cmake -D CMAKE_BUILD_TYPE=RELEASE \ -D CMAKE_INSTALL_PREFIX=/usr/local ..

5、正式编译，sudo make。这步编译大概需要一个小时左右，如不出现error，waring等报错信息则代表编译成功。

七、安装opencv

1、安装 sudo make install

2、更新动态链接库 sudo ldconfig

至此，opencv安装成功

八、测试代码

import cv2

def main():
    cap = cv2.VideoCapture(0)
    while(1):
        ret,frame =cap.read()
        cv2.imshow("name",frame)
        if cv2.waitKey(1)&0XFF == ord('q'):
            break
    cap.release()
    cap.destroyAllWindows()

if __name__ == '__main__':
    main()

九、部分问题解决

1、摄像无法打开

如上述代码报错误
error: (-215) size.width>0 && size.height>0 in function cv::imshow
。

首先检查摄像头是否正常工作。可在Linux的etc配置目录修改一下配置文件，如下。

sudo nano /etc/modules
// 进入编辑界面后，在末尾添加输入
snd-bcm2835
bcm2835-v4l2

可以利用树莓派配置工具打开摄像头模块

输入sudo raspi-config

进入页面如下

选择camera，再选择enable，即可

总结

Linux环境下的OpenCV环境配置并不复杂，需要了解一定的Linux文件操作基础以及部分的Linux软件安装方法。
